# Projeto Site Restaurante NossaMesa Bootstrap

O NossaMesa é um projeto web de um restaurante fictício com intuito de colocar em prática os conhecimentos adquiridos em HTML, CSS, Bootstrap. Com isso, foram utilizados componentes avançados do CSS e Bootstrap como as transições, cards, modal e carrousel de imagens. Para acessar o website, basta acessar o link abaixo:

https://arturbarbuzano.github.io/projeto-site-restaurante-bootstrap/
